flask-生成excel下载文件

发表于 2016-12-03 | 分类于 flask

需求背景

在flask项目中， 查询数据库， 并生成Excel下载文件。

问题

该需求涉及到两个问题：

  • 如何生成excel文件？

  • flask怎么样生成下载文件？

解决：

tablib库

使用tablib库生成excel文件， 首先安装：

pip install  tablib

然后， 使用；样例如下：

import tablib  
headers = (u"列名-1", u"列名-2", u"列名-3")
rows = [
    (u"x", u"xx", u"xxx"),
    (u"y", u"yy", u"yyy"),
]
data = tablib.Dataset(*rows, headers=headers)

然后就可以通过下面这种方式得到各种格式的数据了。

data.xlsx  
data.xls  
data.ods  
data.json  
data.yaml  
data.csv  
data.tsv  
data.html

flask生成下载

已经能生成excel文件了， 怎么样能生成下载文件呢？只需要：

from flask import make_response
resp = make_response(data.xlsx)
resp.headers["Content-Disposition"] = "attachment; filename=文件名.xlsx"
return resp

即可。

Flask生成下载确实简单多了。

注意

使用xls格式的话，最大下载行数为65536；要取消这个限制，导出文件为**.xlsx**格式即可。
