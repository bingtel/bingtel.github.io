python2.7数据模型之函数对象

发表于 2017-01-04

函数对象是Python语言中的一级公民，具有一些特殊属性，包括：

__doc__ (可通过赋值修改)
__name__ (可通过赋值修改)
__module__ (可通过赋值修改)
__defaults__ (可通过赋值修改)
__code__ (可通过赋值修改)
__globals__ (只读)
__dict__ (可通过赋值修改)
__closure__  (只读)

下面举个例子分别介绍一些。

首先，定义一个函数(装饰器)：

那么，

__name__返回函数的名称：

>>> log.__name__
'log'

__doc__返回函数的注释:

>>> log.__doc__
'log\xe8\xa3\x85\xe9\xa5\xb0\xe5\x99\xa8\n    '
>>> print log.__doc__
log装饰器

__module__返回函数所在的模块：

>>> log.__module__
'__main__'

__defaults__返回函数参数的默认值(如果参数中包含关键字参数，则返回一个tuple, 每个元素的是参数的默认值；否则，返回None)：

>>> log.__defaults__

__code__返回函数的代码对象：

>>> log.__code__
<code object log at 0x7fbdd326d130, file "<stdin>", line 1>

globals指向了包含函数全局变量的字典：

{'__builtins__': <module '__builtin__' (built-in)>, 
 '__name__': '__main__',
 'log': <function log at 0x7fbdd3281668>,
 '__doc__': None, '__package__': None}

__dict__返回一个包含函数对象属性的字典：

>>> log.__dict__
{}
>>> log.a=1
>>> log.__dict__
{'a': 1}

__closure__返回None或者一个包含闭包函数的tuple：

>>> log.__closure__
>>> def f():
...     print 'f() called'
... 
>>> t=log(f)
>>> t()
begin
f() called
done
>>> t.__closure__
(<cell at 0x7fbdd32874b0: function object at 0x7fbdd32815f0>,)
